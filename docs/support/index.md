# 支持状态

目前仅适配 **HUAWEI FreeBuds 3**。如需适配更多耳机，可以加入 QQ 群 `1022359908`。

## 已支持功能

| 功能 | 出现位置 |
| --- | --- |
| 左耳、右耳和充电盒电量 | HuaweiPods、系统蓝牙页 |
| 主动降噪开关 | HuaweiPods、系统蓝牙页、超级岛 / 弹窗 |
| 空间降噪方向圆盘 | HuaweiPods、系统蓝牙页 |
| 超级岛与连接弹窗 | HyperOS 系统界面 |
| 融合设备中心显示与流转 | 融合设备中心 |

功能是否出现可能受到 HyperOS 版本、系统组件版本和 LSPosed 作用域影响。
